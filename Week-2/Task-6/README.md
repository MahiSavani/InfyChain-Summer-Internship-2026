# Task 6 — Web Scraping: Data Collection from Online Sources

**Category:** Data Collection

##  Objective
Programmatically collect structured data from an online source to build a custom
dataset for later analysis.

##  Work Done
- Scraped structured data from a web source using **BeautifulSoup** and **Requests**
- Parsed and cleaned the scraped HTML content into a tabular structure
- Stored the results as a **CSV** file for downstream analysis

##  Result
A custom, reproducible dataset built entirely from web-scraped data, ready for
trend/insight analysis in Task 7.

##  Tools & Libraries
Python, BeautifulSoup, Requests, Pandas
