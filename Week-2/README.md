# Week 2 — Dashboarding & Web Scraping

**Project Title:** Dashboard Making using Power BI & Dataset making using Web Scraping

##  Overview
Week 2 shifted focus from notebook-based analysis to business intelligence tooling
and data acquisition — building an interactive Power BI dashboard and collecting a
fresh dataset from the web via scraping.
