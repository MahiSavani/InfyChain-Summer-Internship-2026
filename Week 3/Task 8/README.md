# Task 8 — Feature Engineering on Real-World Datasets

**Category:** Feature Engineering

##  Objective
Apply feature engineering techniques to real-world datasets to prepare them for
machine learning modeling.

##  Work Done
- Applied **encoding** techniques to categorical variables
- Applied **scaling** to numeric features (e.g., StandardScaler)
- Performed **feature selection** to identify the most relevant predictors
- Created **derived features** from existing columns to improve model signal

##  Result
A feature-engineered dataset with properly encoded, scaled, and derived features,
ready for use in supervised and unsupervised modeling tasks.

##  Tools & Libraries
Python, Pandas, Scikit-learn
