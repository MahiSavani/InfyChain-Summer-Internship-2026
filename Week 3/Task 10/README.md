# Task 10 — Exploring Unsupervised ML Algorithms

**Category:** Unsupervised Machine Learning

##  Objective
Explore unsupervised learning techniques for clustering and pattern discovery in
unlabeled data.

##  Work Done

outputs differ on the same dataset — groundwork later applied to customer
segmentation.
