# Week 1 — Data Cleaning and Visualization

**Intern:** Mahi Savani (24AIML059)
**Project Title:** Data Cleaning and Visualization using various datasets

##  Overview
Week 1 focused on the fundamentals of the data science workflow: cleaning messy
real-world data and communicating patterns through visualization, across four
different Kaggle datasets.

##  Tasks

| Task | Dataset | Focus |
|------|---------|-------|
| [Task 1](../Task-1) | Titanic | Data Cleaning & Preprocessing |
| [Task 2](../Task-2) | Iris | Data Visualization |
| [Task 3](../Task-3) | Netflix | Exploratory Data Analysis (EDA) |
| [Task 4](../Task-4) | Sales Dataset | Data Cleaning & Basic EDA |

### Task 1 — Titanic: Data Cleaning & Preprocessing
- Handled missing values in `Age`, `Cabin`, and `Embarked` columns
- Removed duplicate entries and irrelevant features
- Encoded categorical variables (`Sex`, `Embarked`) using Label Encoding

### Task 2 — Iris: Data Visualization
- Created bar charts, histograms, and scatter plots using Matplotlib & Seaborn
- Visualized feature distributions and species-wise comparisons
- Used pair plots to understand inter-feature relationships

### Task 3 — Netflix: Exploratory Data Analysis (EDA)
- Analyzed content type distribution (Movies vs. TV Shows)
- Performed country-wise, genre-wise, and rating-wise analysis
- Visualized trends in content release year and duration
- Identified top contributing countries and most common genres

### Task 4 — Sales Dataset: Data Cleaning & Basic EDA
- Cleaned sales data: handled nulls, corrected data types, removed outliers
- Explored sales trends across categories, regions, and time periods
- Generated summary statistics and visualizations for key metrics
