# Task 4 — Sales Dataset: Data Cleaning & Basic EDA

**Dataset:** Sales Dataset (Kaggle)
**Category:** Data Cleaning & Basic EDA

##  Objective
Clean a raw sales dataset and perform basic exploratory analysis to surface
business-relevant trends.

##  Work Done
- Cleaned sales data: handled nulls, corrected data types, removed outliers
- Explored sales trends across **categories**, **regions**, and **time periods**
- Generated summary statistics and visualizations for key metrics

##  Result
A cleaned, analysis-ready sales dataset with clear category/region/time-based trend
visualizations and summary statistics.

##  Tools & Libraries
Python, Pandas, Matplotlib, Seaborn
