# Task 3 — Netflix Dataset: Exploratory Data Analysis (EDA)

**Dataset:** [Netflix Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows) (~8,800 records, Kaggle)
**Category:** Exploratory Data Analysis

##  Objective
Explore the Netflix content catalog to uncover trends in content type, geography,
genre, and release timing.

##  Work Done
- Handled missing values for `country` and `rating` columns
- Extracted time-based features (`year_added`, `month_added`) from the `date_added` field
- Analyzed **content type distribution** (Movies vs. TV Shows)
- Performed **country-wise**, **genre-wise**, and **rating-wise** analysis
- Visualized trends in **content release year** and **duration**
- Identified top contributing countries and most common genres

##  Result
Identified the peak content-adding year and the top content-producing country, along
with the overall Movies-vs-TV-Shows split and rating distribution.

##  Tools & Libraries
Python, Pandas, Matplotlib, Seaborn
