# Task 2 — Iris Dataset: Data Visualization

**Dataset:** [Iris Dataset](https://www.kaggle.com/datasets/uciml/iris) (150 records, 3 species, Kaggle)
**Category:** Data Visualization

## 🎯 Objective
Build a range of static visualizations to explore feature distributions and
relationships across the three Iris species.

## 🛠️ Work Done
- Created a **grouped bar chart** comparing average measurements per species
- Built **histograms** showing petal length distribution
- Plotted a **scatter plot** of petal length vs. petal width
- Built a full **pairplot** comparing all four features simultaneously
- Generated a **correlation heatmap**
- Used pair plots to understand inter-feature relationships and species-wise comparisons

## ✅ Result
Five distinct visualizations clearly separating the three Iris species by petal
measurements (petal length/width being the most discriminative features).

## 🧰 Tools & Libraries
Python, Pandas, Matplotlib, Seaborn
